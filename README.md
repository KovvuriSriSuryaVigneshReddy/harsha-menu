# Harsha Menu List

### About
**Harsha Menu List** is a modern, responsive web application designed to streamline the restaurant ordering experience. Built with React and powered by Vite, this project allows users to browse menu items seamlessly, search for specific dishes, and manage table orders efficiently. 

### Key Features
*   **Dynamic Menu Display**: Real-time rendering of menu items with availability tracking.
*   **Smart Search**: An intuitive search functionality that filters items by name, category, or serial number.
*   **Order Management**: Integrated features to handle table-specific orders, including parcel order support and status notifications.
*   **Performance Focused**: Optimized build using Vite for fast load times and a smooth user experience.

### Tech Stack
*   **Frontend**: React, JavaScript (JSX), CSS.
*   **Build Tool**: Vite.
*   **Data Handling**: Custom dataset management for menu items.

### How to Run Locally
1.  **Clone the repository**:
    ```bash
    git clone [https://github.com/KovvuriSriSuryaVigneshReddy/harsha-menu-list.git](https://github.com/KovvuriSriSuryaVigneshReddy/harsha-menu-list.git)
    ```
2.  **Navigate to the project directory**:
    ```bash
    cd harsha-menu-list
    ```
3.  **Install dependencies**:
    ```bash
    npm install
    ```
4.  **Start the development server**:
    ```bash
    npm run dev
    ```
